# Apps Script backend

This folder holds `Code.gs`, the server-side logic for the assessment portal.

**It is not deployed from this repo.** You paste it into a Google Apps Script project that is
container-bound to your Google Sheet. Full steps are in the root [README](../README.md#2-google-sheet--apps-script-the-backend).

Quick version:
1. Google Sheet → **Extensions → Apps Script**.
2. Paste `Code.gs` here into the project's `Code.gs`; set `CLIENT_ID`.
3. Add a second script file `answerKey.gs` and paste `build-output/answerKey.gs` (generated by `npm run build`).
4. Run `initSheet` once. Then **Deploy → Web app** (Execute as: Me · Access: Anyone).
5. Redeploy a **new version** any time `Code.gs` or `answerKey.gs` changes.
